Fibonacci Calculator

Fibonacci Calculator - a Kubernetes-based application that calculates Fibonacci values

  • Built out a React front end that uses Axios to make requests to an Express server
  • Created a Redis client to calculate fibonacci values and cache the results
  • Connected a Postgres database for long term storage of input values
  • Configured Dockerfiles to generate containers for each service using Docker-compose
  • Configured a Kubernetes Cluster using an Ingress Service for exposure to outside traffic and ClusterIP Services for internal access to the client, server, Redis and Postgres deployment pods
  • Created a Persistent Volume Claim for the Postgres instance
  • Set up CI/CD through Github using TravisCI to automate testing and deployment
  • Set up project on Google Cloud and created cluster using Kubernetes Engine
  • Installed Cert Manager into cluster and configured issuer to set up HTTPS certificates using Let'sEncrypt

Back End Scaling Project

BookShelf - A searchable book finding app providing description, synopsis, reviews, ratings, and author details

  • Re-designed the back end of the reviews feature in a legacy code application built using React, Node, Express and MySQL in order to handle 10M users
  • Worked closely with a team of engineers using Agile methodologies, performing daily stand-ups and regular code reviews through Git version control
  • Authored NodeJS scripts for generating 100M fake records of book reviews for testing purposes (≈10 reviews/user)
  • Designed database schemas for both MongoDB and PostgreSQL
  • Performed benchmark testing of identical queries to each database to verify the performance on large data sets and chose PostgreSQL due to a 24% faster response time
  • Deployed application to AWS on multiple EC2 instances and migrated the reviews data from a local CSV file onto the remote database
  • Conducted real-time performance testing using New Relic and reduced latency by vertically scaling database instances

Integrated circuit layout

IC layout is the representation of an integrated circuit in terms of planar geometric shapes which correspond to the patterns of metal, oxide, or semiconductor layers that make up the components of the integrated circuit.

These are some of the IC components that I created using a program called Cadence Virtuoso as part of my Electronic Graphics Computer-Aided Design curriculum at ACC. Learning about how these components come together to generate the logic that defines our technological world has been an essential stepping stone on my way to discovering my passion for software engineering as well as sparking my interest in IoT technology.
